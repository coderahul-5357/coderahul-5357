## What I'm building

**🌾 KrishiVeda** — a cloud-native agricultural management platform
A multi-role system (admin, vendor, consultant) running on Cloudflare Workers with Hono.js, Prisma, and PostgreSQL. Handles vendor onboarding, marketplace management, dual-namespace JWT auth, RBAC across five roles, and audit logging. This is the system I spend most of my working hours on — migrated it from a traditional Node.js backend to the Workers runtime, which meant replacing `pg`, `jsonwebtoken`, and `argon2` with Workers-compatible alternatives and wiring up Hyperdrive for the Postgres TCP connection.

**🫀 Deep learning for cardiovascular disease detection** *(research, submitted to Frontiers in Cardiovascular Medicine)*
A hybrid EfficientNet-B3 + BiLSTM + Multi-Head Attention + Transformer architecture trained on the EchoNet-Dynamic and CAMUS echocardiography datasets. The goal: classify cardiac conditions from ultrasound video sequences. This is my M.Tech thesis work.

**🗣️ LSTM-based machine translation (English → French)**
Sequence-to-sequence encoder-decoder built in TensorFlow, evaluated with BLEU scores. Covers the full pipeline — tokenization, embeddings, padding, training, evaluation.

**✍️ NLP text auto-correction system**
Probabilistic language model for contextual spelling/grammar correction, with custom error-detection and accuracy benchmarking.
